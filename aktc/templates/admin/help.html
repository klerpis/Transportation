SCHEDULE SETUP SYSTEM

This guide follows a strict procedure on how to successfully setup trips for a specified amount of time (e.g 2 days, a week, month, 40days, year)

Step 1: Time Setup

  • Departure Time: Create Several Time interval
    e.g [8:00am, 11:00am, 12:00pm, 2:00pm, 4:30pm]
  • Daily Schedule: Pick Multiple Departure times Created to create a cluster for a single day e.g [Wed: 8:00am, 2:00pm, 4:30pm]

    • some days may not carry the same departure schedule as others,
    e.g sunday may only have one trip in the morning, while monday could have multiple trips in a day (up to 5 trips). so the cluster should reflect the number of trip times in a day

Step 2: Location Setup

  • Location: add various locations and the relevant data

    • Fill in the neccessary information for the various location addresses
    like State, Local Government, Bus stop, street
    as well as geographical information like longitude and latitude (leave empty if unknown)

  • Routes: setup different routes

    • A route is a connection (journey) between two locations
    e.g AkwaIbom (Uyo) -> Cross River (Calabar) when adding, an option to immediately add a reverse route exist
    a checkbox - "Add reverse"
    this add the Cross River (Calabar) -> AkwaIbom (Uyo) as well in one go
    Fare and the average Journey length are other needed data inputs

Step 3: Generate Trips Using Schedule

Trips are generated automatically based on data from the previous steps

  • Week Day Schedule:

    • - Input a start date

    - input an end date (optional)

    - input the daily time cluster (Daily Schedule) you wish the following dates to follow

    - Choose the Routes you wish to generate this Trips for (multiple routes can be pick at a time: Recommended)

    - Save and Trips are Auto generated for those dates and time intervals
